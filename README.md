# SellerAgent AI

Drupal 11 + React 19 SPA for Amazon seller advertising automation.

- **Backend**: Drupal 11 (PHP 8.3, MariaDB 11) with custom API modules
- **Frontend**: React 19 + Vite + Tailwind CSS + shadcn/ui
- **Infrastructure**: Docker Compose

## Prerequisites

- Docker & Docker Compose
- Node.js 20+ (with npm)
- Composer 2+

## Setup (Fresh Server)

```bash
# 1. Clone the repo
git clone git@github.com:phpsrajiv/amazon-sites.git
cd amazon-sites

# 2. Create .env from example and update credentials
cp .env.example .env
# Edit .env with production values (DB passwords, hash salt, etc.)

# 3. Install Drupal dependencies
cd seller-backend/drupal
composer install
cd ../..

# 4. Install frontend dependencies and build
cd seller-frontend
npm install
npm run build
cd ..

# 5. Start Docker containers
docker compose up -d

# 6. Install Drupal site using minimal profile with exported config
docker exec selleragent-drupal drush site:install minimal --existing-config -y

# 7. Import content from the exported zip
docker exec selleragent-drupal drush content:import /opt/drupal/web/scs-export/content-bulk-export-15_03_2026-05_07.zip -y

# 8. Rebuild cache
docker exec selleragent-drupal drush cr
```

## SEO & Marketing

- **Meta tags**: Managed via `react-helmet-async` — each page sets title, description, Open Graph, and Twitter Card tags dynamically
- **Structured data**: JSON-LD for Organization (homepage) and BlogPosting (blog posts)
- **Sitemap**: Generated by Drupal's `simple_sitemap` module at `/sitemap.xml`, proxied through Vite
- **robots.txt**: Located at `seller-frontend/public/robots.txt`
- **Google Tag Manager**: Placeholder in `index.html` (replace `GTM-XXXXXXX` with your container ID)
- **SPA page tracking**: `usePageTracking` hook pushes virtual pageviews to GTM dataLayer on route changes

See [drupal-content-strategy.md](drupal-content-strategy.md) Section 22 for full SEO documentation.

## Development

```bash
# Start frontend dev server (with hot reload)
cd seller-frontend
npm run dev

# Drupal cache rebuild
docker exec selleragent-drupal drush cr

# Export Drupal config after changes
docker exec selleragent-drupal drush cex -y

# Export content
docker exec selleragent-drupal drush content:export --all-content --assets -y
```
